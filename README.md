# 🚀 YOLOv8 Object Detection Project

## 📌 Overview
This project implements an **end-to-end Object Detection system** using YOLOv8.  
It can detect objects in **images, videos, and live webcam streams** and save annotated outputs.

---

## 🎯 Features
- ✅ Train custom object detection model (YOLOv8)
- ✅ Evaluate using mAP, Precision, Recall
- ✅ Detect objects in:
  - Images
  - Videos
  - Webcam
- ✅ Save annotated outputs automatically
- ✅ Clean and modular project structure

---

## 🧠 Concepts Covered
- Bounding Boxes
- IoU (Intersection over Union)
- Confidence Score
- Non-Max Suppression (NMS)
- mAP (Mean Average Precision)
